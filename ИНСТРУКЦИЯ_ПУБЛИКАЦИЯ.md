# Инструкция: размещение материалов на сайте zemlya-bez-torgov.netlify.app

## Архитектура

| Файл / папка | Назначение |
|---|---|
| `terra-ufa-bez-torgov/` | Источник: markdown-драфты каждого материала |
| `zemlya_bez_torgov.html` | Рабочий HTML сайта — редактируется вручную |
| `_deploy/index.html` | Генерируется автоматически при деплое (не трогать) |
| `deploy.ps1` | Скрипт публикации: копирует HTML → `_deploy`, деплоит на Netlify |
| `post/index.md` | Индекс всех материалов (хронологический список) |

Сайт раздаётся из `_deploy/index.html`. Скрипт `deploy.ps1` сам копирует туда `zemlya_bez_torgov.html` — редактировать нужно только `zemlya_bez_torgov.html`.

## Часть 1. Материал уже есть в папке `terra-ufa-bez-torgov/`

### Шаг 1. Открыть markdown-файл и взять данные

Каждый файл в `terra-ufa-bez-torgov/` содержит фронтматтер с полями:

```yaml
---
num: 10
сайт: terra_ufa_bez_torgov
статус: драфт           # → поменять на "опубликовано" после деплоя
дата: 2025-05-19
источник: РБК Уфа
url: https://...
теги: ...
---
```

Нужны: `дата`, `источник`, `url`, заголовок (H1) и текст тела.

### Шаг 2. Добавить карточку в `zemlya_bez_torgov.html`

Найти раздел `<!-- ═══ КАРТОЧКА N ═══ -->` — карточки расположены хронологически. Вставить новую **перед** последней нехронологической карточкой (сейчас — карточка «Картотека АД · 18ААС», она всегда идёт последней).

#### Шаблон карточки — внешнее СМИ

```html
<!-- ═══ КАРТОЧКА N ═══ -->
<article class="media-pub-card">
  <div class="media-pub-source">
    <span class="media-pub-outlet">НАЗВАНИЕ ИСТОЧНИКА</span>
    <span class="media-pub-date">ДД месяц ГГГГ</span>
  </div>
  <div class="media-pub-headline"><a href="URL_ОРИГИНАЛА" target="_blank">ЗАГОЛОВОК</a></div>
  <p class="media-pub-excerpt">ТЕКСТ ВЫЖИМКИ (3–5 предложений, факты без оценок)</p>
  <div class="media-pub-foot">
    <a href="URL_ОРИГИНАЛА" class="media-pub-link" target="_blank">Материал в источнике</a>
  </div>
</article>
```

#### Особые карточки

Карточка «Судебный документ / без внешней ссылки» (как 18ААС):
```html
<article class="media-pub-card" style="border-top-color: var(--gold);">
  <div class="media-pub-source">
    <span class="media-pub-outlet" style="color:#8A6200; border-color:#8A6200;">ИСТОЧНИК</span>
    <span class="media-pub-date">ДД месяц ГГГГ</span>
  </div>
  <div class="media-pub-headline">ЗАГОЛОВОК (без ссылки)</div>
  <p class="media-pub-excerpt">ТЕКСТ</p>
  <div class="media-pub-foot">
    <span class="media-pub-link" style="cursor:default; color:var(--gold);">РЕКВИЗИТЫ ДОКУМЕНТА</span>
  </div>
</article>
```

### Шаг 3. Обновить счётчики и тикер (опционально)

Если добавляется материал о новом объекте или значимо меняется счёт — обновить:
- Тикер (строка `dateline-ticker`) — добавить факт
- Блок «Объекты расследования» (`inst-count`) — обновить число материалов

### Шаг 4. Обновить `post/index.md`

Добавить строку в таблицу:
```
| N | ГГГГ-ММ-ДД | Источник | [имя_файла.md](имя_файла.md) |
```

### Шаг 5. Поменять статус в markdown-файле

В `terra-ufa-bez-torgov/имя_файла.md` поменять `статус: драфт` → `статус: опубликовано`.

### Шаг 6. Задеплоить

```powershell
cd "c:\Users\user\.claude\projects\crinj_metod\Uzechim\site_bez_torgov"
.\deploy.ps1
```

Скрипт сам скопирует `zemlya_bez_torgov.html` в `_deploy/index.html` и опубликует. После завершения — проверить на уникальном URL деплоя (он выводится в конце).

## Часть 2. Новый материал — создать драфт, затем опубликовать

### Шаг 1. Создать файл в `terra-ufa-bez-torgov/`

Имя файла: `[slug].md`, где slug = транслит основных слов через дефис.

Шаблон:

```markdown
---
num: N                          # следующий по порядку
сайт: terra_ufa_bez_torgov
статус: драфт
дата: ГГГГ-ММ-ДД               # дата оригинальной публикации
источник: Название СМИ
url: https://...
автор: Имя Фамилия             # если указан
теги: ключевые, слова, через, запятую
---

# Заголовок материала

Текст пересказа / выжимки. Ориентир: 4–6 абзацев, факты с цифрами, без оценок.

## О компании (если нужен блок)

...

---

**Источник:** Название СМИ. «Заголовок оригинала». ДД месяц ГГГГ г. URL

---

## SEO-метаданные

- **Title (browser):** Заголовок для title-тега
- **Meta description:** 140–160 символов
- **Slug:** имя-файла-без-расширения

## На полях (редактору)

- Что проверить / уточнить / добавить в расширенную версию
```

### Шаг 2. Написать текст выжимки

Принципы:
- Факты с цифрами (площадь, кадастровая стоимость, выручка, даты)
- Связь с расследованием — упомянуть Горбунова / Иванова / УЗЭХИМ, если фигурируют
- Цитаты экспертов — если есть, вставить дословно
- Последнее предложение — вопрос или противоречие (системное наблюдение)
- Без буквы «ё», без длинного тире

### Шаг 3. Добавить карточку на сайт и задеплоить

Далее — по Части 1, шаги 2–6.

## Хронологический порядок карточек

Карточки в `zemlya_bez_torgov.html` идут по дате публикации источника. Единственное исключение — «закреплённые» карточки с судебными решениями (золотая рамка), они всегда в конце.

Чтобы найти правильное место для вставки — искать комментарий с датой соседних карточек:
```
<!-- ═══ КАРТОЧКА 9 ═══ --> (14 апреля 2025)
← сюда вставлять материал от мая 2025
<!-- ═══ КАРТОЧКА 10 ═══ --> (19 мая 2025) ← уже добавлена
```
