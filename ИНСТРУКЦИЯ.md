# Управление сайтом «Земля·Уфа·Без Торгов»

**Сайт:** https://zemlya-bez-torgov.netlify.app  
**Репозиторий:** https://github.com/Razum-hub/zemlya-bez-torgov  
**Главный файл:** `zemlya_bez_torgov.html`

---

## Как задеплоить изменения

1. Отредактировать `zemlya_bez_torgov.html` в VS Code
2. Открыть терминал в папке `site_bez_torgov`
3. Запустить:

```powershell
powershell -File deploy.ps1
```

Сайт обновится через 5–10 секунд.

## Добавить событие в хронику

Найти блок `dateline-ticker` (строка ~1257), добавить строку:

```html
<span>ДД.ММ.ГГГГ · Текст события</span>
```

---

## Обновить срочную новость

Найти `alert-band` (строка ~1249):

```html
<span class="alert-band-label">Тема · ДД.ММ.ГГГГ</span>
<span class="alert-band-text">Текст новости одним абзацем.</span>
```

---

## Добавить карточку материала

Найти нужный раздел, скопировать и вставить блок:

```html
<div class="r-cell flex">
  <div class="art-tag">Тег · Метка</div>
  <div class="art-h3"><a href="#">Заголовок материала</a></div>
  <p class="art-body" style="font-size:0.88rem;">Текст аннотации одним абзацем.</p>
  <div class="art-foot">
    <span class="art-date">ДД.ММ.ГГГГ</span>
    <a href="#" class="art-link">Читать →</a>
  </div>
</div>
```

Для тёмной карточки (главный материал) добавить класс `dark`:

```html
<div class="r-cell dark flex">
```

---

## Обновить SEO при добавлении важного события

Строка 12 — `meta name="description"`:

```html
<meta name="description" content="Новое описание с актуальными фактами.">
```

Строка 21 — `og:description` (для соцсетей):

```html
<meta property="og:description" content="Краткое описание для превью.">
```


## Подключить собственный домен

1. Netlify → проект `zemlya-bez-torgov` → **Domain management**
2. **Add a domain** → вписать домен
3. Прописать у регистратора NS-серверы Netlify или CNAME-запись
